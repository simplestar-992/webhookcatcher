# WEBHOOKCATCHER

**Debug Webhooks with a Local Endpoint**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/webhookcatcher.git
cd webhookcatcher
go build -o webhookcatcher .
```

## Usage

```bash
./webhookcatcher --help
```
